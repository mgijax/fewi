Logic Summary

Querying in Text Fields

Description of logic used in each of the text field inputs when a query is submitted:

Nomenclature

  • Performs a tokenized query against Solr index.
  • Initial and trailing wildcard symbols (asterisk) are supported; "internal" asterisks are treated literally.
  • Tokens are combined with Boolean AND.
  • Does not include search against allele synonyms.
  • Attempts to recreate, as much as possible, previous code's default sorting if Nomen field is submitted (except now using Solr field boosts):
    MRK_label table, April 2014, additional Python code sorted "exact" matches ahead of matches contained within text strings.

GO and Protein Domain (InterPro)

  • Performs a tokenized query against Solr index.
  • Performs stemming.
  • Supports initial and trailing wildcards.
  • Supports quotation marks for phrase matching.
  • Supports parentheses for grouping.
  • Supports Booleans AND, OR, NOT, AND NOT. Default is OR.
  • The 3 GO DAG root node terms are excluded.

Pheno/Disease

  • Performs a tokenized query against Solr index.
  • Initial and trailing wildcard symbols (asterisk) are supported.
  • Supports quotation marks for phrase matching.
  • Supports parentheses for grouping.
  • Supports Booleans AND, OR, NOT, AND NOT. Default is OR.
  • Returns only genes having annotated alleles in One Marker Genotypes (OMG) or "Simple" genotypes: 2-marker Conditional genotypes and 2-marker genotypes involving a Transgenic Reporter.
    • EXCLUDES markers for annotated alleles present only in Complex genotypes (unlike the Allele Query): genotypes having 3 or more markers; genotypes having 2 markers that are not Conditional and do not contain a Transgenic Reporter.
    • EXCLUDES markers of slash alleles (unlike Allele Query) - examples:
      • MP:0004800 returns Lta/Tnf<tm1Eug> twice and also Lta/Tnf<tm1Jods> twice on the Allele Query, but neither Lta nor Tnf on the Marker Query.
      • MP:0004984 returns Bglap/Bglap2<tm1Kry> twice on the Allele Query, but neither Bglap nor Bglap2 on the Marker Query.
  • Excludes Recombinase markers in Conditional genotypes.
  • Excludes Transgenic Reporter markers.
  • Searches for matches in Mapping Experiment Notes when the Mapping Experiment type is TEXT-QTL only for markers of type QTL.
